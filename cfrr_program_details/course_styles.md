# Course Delivery Styles

This page explains what each course delivery style means in the CfRR programme.

## Live Coding

Live Coding sessions are taught by coding together with the workshop leader in real time.
You follow along on your own machine, run commands as they are introduced, and ask questions as you go.

## Seminar

Seminar sessions are primarily discussion and concept focused.
They usually use slides, worked examples, and open Q&A to build understanding.

## Surgery

Surgery sessions are ad-hoc problem-solving sessions.
You bring your own questions or blockers and work through them with support from staff and peers.

## Hybrid

Hybrid sessions combine seminar and live coding approaches.
Part of the session is concept/discussion based, and part is practical coding together.

## Notes

- Some courses may evolve over time, so check each workshop page for the latest style label.
